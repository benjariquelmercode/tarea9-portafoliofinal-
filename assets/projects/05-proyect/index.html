#Galería #Diseño #Naturaleza #Conceptual #Demo

Propuesta minimalista

Se necesita crear una galería de fotos web, que haga uso de los siguientes conceptos, responsividad, colores parametrizados, modelo de caja de css, transforms, flexbox y grid. La galería debe ser idéntica a la que se muestra en un Demo visto en la pauta solicitada, utilizando Imágenes adjuntas, ubicaciones predeterminadas, en cada una de las Pantallas. Sólo se puede cambiar el color para la caja del título y la barra de navegación.
